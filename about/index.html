About Me

Hi my name is Leon and I’m currently studying computer science at the Christian-Albrechts-University in Kiel, Germany. I have a strong preference for coding in Python and am mainly interested in Backend development and prototyping. I love clean, well structured and simple code and enjoy the aesthetics of such solutions for complex problems. I’m familiar with most of today’s technologies down to the protocol level.

I would describe myself as a Backend Developer, because everything Frontend related doesn’t really catch my attention (even though I know about plain HMTL/CSS/JS). I’m mostly into:

  • Backend development using Python and Flask (sometimes Django) or Java
  • Databases (relational & non-relational)
  • Redis
  • ELK-Stack
  • Microservices and Docker
  • Security
